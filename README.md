# MYBUNTU
Custom Ubuntu Theme

## FONTS

**Window Title**: Lucida MAC Regular - 9 (NoobLabs MACBUNTU)
**Interface**: Myriad Set Pro Semibold - 10
**Document**: Myriad Set Pro Semibold - 10
**Monospaced**: Andale Mono Regular - 9

Scaling Factor: 0.95

* User themes
